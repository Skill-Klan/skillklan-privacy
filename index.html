1. ЗАГАЛЬНІ ПОЛОЖЕННЯ

Ця Політика конфіденційності (далі — «Політика») визначає принципи, порядок та умови обробки персональних даних користувачів мобільного додатку Skill Klan (далі — «Додаток»), який є системою управління навчальними процесами та використовується адміністраторами, менторами та іншими уповноваженими учасниками освітнього середовища.

Адміністрація Додатку (далі — «Ми», «Адміністрація») забезпечує дотримання вимог Закону України «Про захист персональних даних», Регламенту (ЄС) 2016/679 (GDPR) та інших нормативно-правових актів у сфері захисту інформації.

Використовуючи Додаток, Користувач надає згоду на обробку персональних даних згідно з положеннями цієї Політики. У разі незгоди Користувач повинен припинити використання Додатку.

2. КАТЕГОРІЇ ОБРОБЛЮВАНИХ ДАНИХ

2.1 Дані ідентифікації та автентифікації

Обробляються відомості, необхідні для створення, управління й захисту облікового запису Користувача та забезпечення безпечного доступу до Додатку.

2.2 Дані, пов'язані з функціональністю Додатку

Обробляються дані, необхідні для реалізації механізмів управління навчальними процесами, координації взаємодії між учасниками та підтримки роботи освітньої інфраструктури.

2.3 Дані, пов'язані з адміністративно-обліковими процесами

У межах роботи Додатку можуть оброблятися відомості, що відображають:

  • фактичний стан виконання взаємних обов'язків між учасниками системи,
  • підтвердження завершення певних дій,
  • адміністративні статуси,
  • інформацію операційно-організаційного характеру, необхідну для ведення внутрішніх реєстрів та забезпечення коректної роботи облікових механізмів.

Важливо: Ці дані не містять платіжних реквізитів, не ідентифікують банківські або інші засоби здійснення розрахункових дій та не використовуються для надання фінансових послуг.

2.4 Технічні дані

Під час використання Додатку можуть автоматично збиратися технічні параметри пристрою, журнали подій, інформація для діагностики та забезпечення стабільності роботи.

3. ЦІЛІ ТА ПРАВОВІ ПІДСТАВИ ОБРОБКИ

Персональні дані обробляються виключно для:

  • забезпечення роботи функціоналу Додатку,
  • підтримання безпеки та автентифікації,
  • належного виконання взаємних зобов'язань між учасниками системи,
  • ведення адміністративно-облікових процесів,
  • розвитку та оптимізації сервісу,
  • дотримання вимог чинного законодавства.

Правові підстави:

  • згода суб'єкта персональних даних,
  • виконання договору, стороною якого є Користувач,
  • виконання правових зобов'язань Адміністрації,
  • захист життєво важливих інтересів Користувача.

4. ЗБЕРІГАННЯ ТА ЗАХИСТ ДАНИХ

4.1 Локальне зберігання

Частина інформації може зберігатися локально на пристрої з використанням стандартних механізмів безпеки операційної системи.

4.2 Серверна інфраструктура

Основний масив даних зберігається на серверних потужностях, розміщених в Україні та/або інших юрисдикціях, які забезпечують відповідність міжнародним стандартам безпеки.

4.3 Термін зберігання

Дані зберігаються протягом часу, необхідного для виконання цілей обробки або протягом строків, передбачених законодавством. Після цього дані підлягають знищенню або анонімізації.

4.4 Обмеження щодо розрахункових даних

Додаток не використовується для здійснення розрахункових операцій, не містить та не обробляє дані, що дозволяють ідентифікувати засоби розрахунку.

Будь-які відомості, які мають ознаки облікового характеру, використовуються виключно для внутрішньої фіксації фактів завершення відповідних взаємодій, необхідних для функціонування Додатку.

5. ПЕРЕДАЧА ДАНИХ ТРЕТІМ ОСОБАМ

Передача персональних даних може здійснюватися виключно:

  • постачальникам технічних та інфраструктурних послуг,
  • сервісам хостингу,
  • організаціям, які забезпечують роботу компонентів Додатку,
  • іншим суб'єктам, задіяним у наданні функціональних можливостей Додатку.

Передача не передбачає надання інформації, яка може кваліфікуватися як платіжна або дозволяє ідентифікувати розрахункові засоби.

Усі треті особи зобов'язані забезпечити конфіденційність та захист персональних даних згідно із законом.

6. ВИКОРИСТАННЯ БІОМЕТРИЧНИХ ДАНИХ

Адміністрація не має доступу до біометричних шаблонів та не здійснює їх зберігання або передачу.

During App usage, technical device parameters, event logs, diagnostic information and information to ensure stable operation may be automatically collected.

Personal data is processed exclusively for:
