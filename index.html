1. ВСТУП

Skill Klan ("ми", "наш", "додаток") — це CRM система для управління навчальними центрами, призначена для менторів та адміністраторів навчальних закладів. Ми серйозно ставимося до захисту вашої конфіденційності та особистих даних.

Ця Політика конфіденційності описує, які дані ми збираємо, як ми їх використовуємо, зберігаємо та захищаємо. Використовуючи наш додаток, ви погоджуєтеся з цією політикою.

Ми залишаємо за собою право оновлювати цю політику. Про всі зміни ми повідомимо вас через додаток або електронну пошту.

2.1 Контактна інформація

  • Email адреса - використовується для автентифікації та зв'язку з вами

2.3 Фінансова інформація

  • Дані про оплати - інформація про платежі студентів
  • Сума оплат - для відстеження фінансових операцій

2.4 Дані про студентів та менторів

3.1 Функціональність додатку

Ми використовуємо ваші дані для:

  • Автентифікації та авторизації - для входу в додаток та забезпечення безпеки
  • Управління обліковим записом - для збереження ваших налаштувань
  • Відображення інформації - для показу даних про студентів, оплати та договори
  • Відстеження оплат - для моніторингу фінансових операцій

3.3 Безпека

  • Запобігання шахрайству - для виявлення підозрілої активності
  • Захист даних - для забезпечення безпеки вашого облікового запису

4. ЗБЕРІГАННЯ ДАНИХ

4.1 Локальне зберігання

Деякі дані зберігаються локально на вашому пристрої:

  • Токени автентифікації - зберігаються в iOS Keychain (захищене сховище)
  • Налаштування додатку - зберігаються локально
  • PIN-код - зберігається в зашифрованому вигляді локально

4.2 Зберігання на сервері

Основні дані зберігаються на захищеному сервері:

  • Контактна інформація - на захищеному сервері з шифруванням
  • Дані про студентів та менторів - в захищеній базі даних
  • Дані про оплати - в Google Sheets (через API) та на сервері

4.3 Термін зберігання

  • Активні облікові записи - дані зберігаються до видалення облікового запису
  • Видалені облікові записи - дані видаляються протягом 30 днів після видалення
  • Резервні копії - можуть зберігатися до 90 днів для відновлення

Дані про оплати можуть передаватися в Google Sheets через API (https://payments.skillklan.com.ua) для:

SKILL KLAN PRIVACY POLICY

Last Updated: January 2025

1. INTRODUCTION

Skill Klan ("we", "our", "app") is a CRM system for managing educational centers, designed for mentors and administrators of educational institutions. We take the protection of your privacy and personal data seriously.

This Privacy Policy describes what data we collect, how we use, store, and protect it. By using our app, you agree to this policy.

We reserve the right to update this policy. We will notify you of any changes through the app or email.

2. WHAT DATA WE COLLECT

2.1 Contact Information

  • Email address - used for authentication and communication with you
  • Phone number - used for identification and communication

2.2 Identifiers

  • User ID - unique identifier for your account
  • Device ID - device identifier (if collected for security)

2.3 Financial Information

  • Payment data - information about student payments
  • Payment amounts - for tracking financial operations
  • Payment dates - for payment history
Important: We do not collect or store credit card data or other payment information. The app only tracks information about payments that have already been made through other channels.

2.4 Student and Mentor Data

  • Names of students and mentors
  • Student contact information
  • Learning statuses
  • Contract data

2.5 Technical Data

  • IP address - may be collected automatically when using the API
  • Device type and operating system
  • App version

3. HOW WE USE DATA

3.1 App Functionality

We use your data for:

  • Authentication and authorization - for logging into the app and ensuring security
  • Account management - for saving your settings
  • Displaying information - for showing data about students, payments, and contracts
  • Payment tracking - for monitoring financial operations

3.2 Service Improvement

  • Error analytics - for detecting and fixing technical issues
  • Performance improvement - for optimizing app performance

3.3 Security

  • Fraud prevention - for detecting suspicious activity
  • Data protection - for ensuring the security of your account

4. DATA STORAGE

4.1 Local Storage

Some data is stored locally on your device:

  • Authentication tokens - stored in iOS Keychain (secure storage)
  • App settings - stored locally
  • PIN code - stored encrypted locally

4.2 Server Storage

Main data is stored on a secure server:

  • Contact information - on a secure server with encryption
  • Student and mentor data - in a secure database
  • Payment data - in Google Sheets (via API) and on the server

4.3 Retention Period

  • Active accounts - data is stored until account deletion
  • Deleted accounts - data is deleted within 30 days after deletion
  • Backups - may be stored for up to 90 days for recovery

5. DATA SHARING WITH THIRD PARTIES

5.1 Backend API Server

Your data is transmitted to our Backend API server (https://api.skillklan.com.ua) for:

  • Request processing
  • Data storage
  • Authentication

Protection: All data is transmitted through encrypted HTTPS connection.

5.2 Google Sheets API

Payment data may be transmitted to Google Sheets via API (https://payments.skillklan.com.ua) for:

  • Financial data storage
  • Report generation

Protection: Secure API connection with authentication is used.

5.3 Other Third Parties

We DO NOT share your data with:

  • Advertising networks
  • Analytics services for tracking
  • Other companies for marketing

6. BIOMETRIC DATA USAGE

6.1 Face ID / Touch ID

The app uses biometric authentication (Face ID or Touch ID) for:

  • Protecting financial data - for accessing the payments section
  • Quick login - as an alternative to PIN code

6.2 Biometric Data Storage

Important: We DO NOT collect or store your biometric data. All biometric data is processed exclusively locally on your device through iOS Secure Enclave. We do not have access to this data.

6.3 Alternative

If you do not want to use biometric authentication, you can use a PIN code.

7. NOTIFICATIONS

7.1 Local Notifications

The app uses local notifications for:

  • Payment check reminders - so you don't miss important dates
  • Important event reminders - about contract dates and other events

7.2 Notification Management

You can:

  • Enable/disable notifications - in iOS settings
  • Configure notification types - in app settings

7.3 Data in Notifications

Notifications contain only necessary information and do not transmit personal data to third parties.
