1. ЗАГАЛЬНІ ПОЛОЖЕННЯ

Настояща Політика конфіденційності (далі — "Політика") визначає порядок обробки та захисту персональних даних користувачів мобільного додатку Skill Klan (далі — "Додаток"), який є системою управління навчальними процесами та призначений для використання менторами та адміністраторами навчальних закладів.

Адміністрація Додатку (далі — "Ми", "Адміністрація") зобов'язується дотримуватися вимог чинного законодавства України та міжнародних стандартів щодо захисту персональних даних, зокрема Закона України "Про захист персональних даних" та Регламенту (ЄС) 2016/679 (GDPR).

Використовуючи Додаток, Користувач надає свою згоду на обробку персональних даних відповідно до умов, викладених у цій Політиці. У разі незгоди з умовами Політики, Користувач зобов'язаний припинити використання Додатку.

2. КАТЕГОРІЇ ОБРОБЛЮВАНИХ ДАНИХ

В процесі надання послуг через Додаток можуть оброблятися наступні категорії персональних даних:

2.1 Дані, необхідні для ідентифікації та автентифікації

Для забезпечення функціонування системи автентифікації та управління обліковими записами обробляються дані, що дозволяють ідентифікувати Користувача та забезпечити безпеку доступу до Додатку.

2.2 Дані, пов'язані з функціональністю Додатку

Для реалізації основного функціоналу Додатку обробляються дані, необхідні для управління навчальними процесами, ведення обліку та забезпечення комунікації між учасниками навчального процесу.

2.3 Дані, пов'язані з фінансовими операціями

В межах функціоналу Додатку можуть оброблятися дані, пов'язані з відстеженням та обліком фінансових операцій, необхідні для ведення фінансової звітності та контролю виконання зобов'язань.

Важливо: Додаток не здійснює обробку платіжних реквізитів та не має доступу до даних платіжних карток. Обробці підлягають лише дані про факт виконання фінансових операцій, отримані з дозволених джерел.

2.4 Технічні дані

В процесі використання Додатку автоматично можуть збиратися технічні дані, необхідні для забезпечення стабільної роботи системи, захисту від несанкціонованого доступу та покращення якості наданих послуг.

3. ЦІЛІ ТА ПІДСТАВИ ОБРОБКИ ДАНИХ

Обробка персональних даних здійснюється виключно в межах та для досягнення наступних цілей:

  • Надання послуг, передбачених функціоналом Додатку
  • Забезпечення безпеки та запобігання несанкціонованому доступу
  • Виконання зобов'язань перед Користувачами
  • Дотримання вимог чинного законодавства
  • Покращення якості наданих послуг та розвиток функціоналу Додатку

Правовими підставами для обробки персональних даних є:

  • Згода суб'єкта персональних даних на обробку
  • Виконання договору, стороною якого є суб'єкт персональних даних
  • Виконання покладених на Адміністрацію правових зобов'язань
  • Захист життєво важливих інтересів суб'єкта персональних даних

4. СПОСОБИ ТА МІСЦЯ ЗБЕРІГАННЯ ДАНИХ

4.1 Локальне зберігання

Частина даних, необхідних для функціонування Додатку, може зберігатися безпосередньо на пристрої Користувача з використанням механізмів захисту, передбачених операційною системою пристрою.

4.2 Зберігання на серверній інфраструктурі

Основна частина оброблюваних даних зберігається на серверній інфраструктурі, розташованій на території України та/або інших юрисдикцій, з дотриманням вимог щодо захисту інформації та відповідності міжнародним стандартам безпеки.

4.3 Терміни зберігання

Персональні дані зберігаються протягом терміну, необхідного для досягнення цілей обробки, або протягом терміну, встановленого чинним законодавством. Після досягнення цілей обробки або закінчення встановленого терміну персональні дані підлягають знищенню або анонімізації відповідно до вимог законодавства.

5. ПЕРЕДАЧА ДАНИХ ТРЕТІМ ОСОБАМ

В межах необхідності для забезпечення функціонування Додатку та надання послуг, персональні дані можуть передаватися третім особам, які надають технічну підтримку, хостинг послуги або інші послуги, пов'язані з роботою Додатку.

Всі треті особи, яким передаються персональні дані, зобов'язуються дотримуватися вимог чинного законодавства щодо захисту персональних даних та застосовувати належні технічні та організаційні заходи для захисту інформації.

Передача персональних даних здійснюється виключно в межах, необхідних для досягнення заявлених цілей обробки, та не передбачає передачу даних рекламним мережам, аналітичним сервісам для відстеження активності Користувачів або іншим особам для маркетингових цілей.

6. ВИКОРИСТАННЯ БІОМЕТРИЧНИХ ДАНИХ

Додаток може використовувати біометричні технології автентифікації, передбачені операційною системою пристрою, для підвищення рівня безпеки доступу до окремих функцій Додатку.

Важливо: Всі біометричні дані обробляються виключно локально на пристрої Користувача з використанням спеціалізованого апаратного забезпечення безпеки. Адміністрація Додатку не має доступу до біометричних даних та не здійснює їх збір, зберігання або передачу.

Користувач має право відмовитися від використання біометричної автентифікації та використовувати альтернативні методи захисту доступу.

Додаток може використовувати механізми сповіщень, передбачені операційною системою пристрою, для інформування Користувача про важливі події та нагадування, пов'язані з використанням Додатку.

Користувач має право налаштувати параметри сповіщень або повністю відключити їх через налаштування операційної системи пристрою.

Сповіщення не містять персональних даних та не передаються третім особам.

  • На знищення своїх персональних даних, обробка яких здійснюється з порушенням вимог, встановлених законом
  • Оскаржити дії або бездіяльність розпорядника при обробці та захисті своїх персональних даних

2.1 Data Required for Identification and Authentication

To ensure the functioning of the authentication system and account management, data that allows identification of the User and ensures security of access to the App is processed.

2.2 Data Related to App Functionality

To implement the main functionality of the App, data necessary for managing educational processes, maintaining records, and ensuring communication between participants in the educational process is processed.

Within the App's functionality, data related to tracking and accounting of financial operations, necessary for maintaining financial records and monitoring fulfillment of obligations, may be processed.

In the process of using the App, technical data necessary for ensuring stable system operation, protection against unauthorized access, and improving the quality of services provided may be automatically collected.

3. PURPOSES AND LEGAL BASIS FOR DATA PROCESSING

Personal data processing is carried out exclusively within and for achieving the following purposes:

  • Providing services provided by the App's functionality

  • Improving the quality of services provided and developing the App's functionality

The legal bases for personal data processing are:

  • Consent of the personal data subject to processing

4.1 Local Storage

Part of the data necessary for the App's functioning may be stored directly on the User's device using protection mechanisms provided by the device's operating system.

4.2 Storage on Server Infrastructure

The main part of processed data is stored on server infrastructure located in Ukraine and/or other jurisdictions, in compliance with information protection requirements and international security standards.

Personal data is stored for the period necessary to achieve the purposes of processing, or for the period established by current legislation.
