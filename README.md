# Skill Klan Privacy Policy

Цей репозиторій містить Privacy Policy для додатку Skill Klan, опубліковану на GitHub Pages.

## Посилання

Privacy Policy доступна за адресою:
- **GitHub Pages:** https://skill-klan.github.io/skillklan-privacy/
- **Альтернативний URL:** https://skill-klan.github.io/skillklan-privacy/index.html

## Використання

Ця Privacy Policy використовується для:
- Додавання в App Store Connect як Privacy Policy URL
- Відображення користувачам додатку
- Відповідності вимогам Apple App Store

## Оновлення

При оновленні Privacy Policy:
1. Внесіть зміни в `index.html`
2. Оновіть дату останнього оновлення
3. Закомітьте та запуште зміни
4. GitHub Pages автоматично оновить сторінку

## Структура

- `index.html` - HTML сторінка з Privacy Policy (українською та англійською)
- `README.md` - цей файл

## Налаштування GitHub Pages

1. Перейдіть в Settings репозиторію
2. Знайдіть розділ "Pages"
3. Виберіть source: "Deploy from a branch"
4. Виберіть branch: "main" (або "master")
5. Виберіть folder: "/ (root)"
6. Натисніть Save

GitHub Pages автоматично опублікує сторінку за адресою:
`https://[username].github.io/skillklan-privacy/`

## Ліцензія

Цей контент є частиною проєкту Skill Klan.
